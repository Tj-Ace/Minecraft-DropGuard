# Drop Guard

Tiny NeoForge `1.21.1` client-side mod that blocks the normal world drop key.

## What it does

- Pressing drop in the world does nothing.
- Pressing drop inside an inventory screen still works normally when you are hovering a slot.

## Notes

- Targets NeoForge `21.1.209` for Minecraft `1.21.1`.
- This is client-only and does not need to be installed on a server.
